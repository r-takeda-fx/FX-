## 1. 目的（Why）

株式投資に興味はあるが「情報が多すぎて何を見ればいいかわからない」という初心者の課題を解決する。複雑な金融情報を**シンプルに整理**し、投資判断の第一歩を支援する。

## 2. ゴール（What）

### MVP（最小構成）
- 市場サマリーダッシュボード（日経平均・米国市場・先物を一画面で）
- 重要ニュースの自動分類・表示（カテゴリ別）
- 初心者向けおすすめ銘柄（静的リスト）

### Nice to Have（後回し）
- ポートフォリオ管理
- プッシュ通知
- ダークモード

## 3. 心理ゴール（Who）

| Before | After |
|--------|-------|
| 「株価とかニュース多すぎて何見ればいいの？」 | 「今日の市場状況、5分で把握できた」 |
| 「FRBとか言われても意味わからない」 | 「このニュースが株価に影響するのね」 |
| 「どの銘柄買えばいいか全然わからない」 | 「初心者向けの銘柄候補がわかった」 |

## 4. 想定障害と対策

| 障害 | 対策 |
|------|------|
| 株価APIは高額 | TradingViewウィジェット（無料）で表示 |
| ニュース収集が複雑 | RSS + RSS2JSON（無料）を活用 |
| おすすめ銘柄の責任問題 | 「投資は自己責任」の免責表示 + 教育目的として提示 |
| 有料API回避 | 全て無料サービスで構成 |

## 5. アクションプラン

| Phase | 内容 | 成果物 |
|-------|------|--------|
| **1. 基盤** | プロジェクト構築、Vite設定 | 動作する骨組み |
| **2. 市場データ** | TradingViewウィジェット埋め込み | ダッシュボード画面 |
| **3. ニュース** | RSS取得・カテゴリ分類 | ニュースセクション |
| **4. 銘柄推薦** | 初心者向け銘柄の静的リスト | おすすめ銘柄セクション |

## 技術構成

```
フロントエンド: HTML + CSS + TypeScript
ビルドツール: Vite
株価表示: TradingViewウィジェット（無料）
ニュース: RSS + RSS2JSON API（無料）
デプロイ: GitHub Pages（無料）
```

### 使用する無料サービス

| サービス | 用途 | 制限 |
|----------|------|------|
| TradingView Widget | 株価チャート・ティッカー | 無料、クレジット表示必須 |
| RSS2JSON | RSSフィード取得 | 無料枠: 10,000リクエスト/日 |
| GitHub Pages | ホスティング | 無料 |
